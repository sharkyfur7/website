The indie web

Then I found out about neocities and the indie web as a whole (circa late 2023) and it made me realize that webdev doesn't have to be all the modern slop with javascript frameworks and whatever else. It made me realize that sites can just be much simpler (though bare HTML and CSS is a bitch to maintain at scale lol).

My site isn't that simple (still not as complex as modern sites though haha) but that is thanks to constant updates since April 2025 (over 200 commits! wow!). I've had other "personal site" projects, however I abandoned or gave up on them. This one seems like it could last a bit longer though...

Should you do all that too?

If you're thinking about making a website, do it. Just put whatever you like on there (like I did with sharks!) Lowkey, all you really need is a bunch of HTML and CSS files. Just experiment and get inspired by other sites, that's pretty much how I get ideas on what to add onto my site.
